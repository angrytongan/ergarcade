pm5-detail

The %concept2% %performance-monitor% (PM) collects various pieces of information about how you are performing in real-time. This application displays all of this information on a single webpage.

Start pm5-detail!

Usage

  • Drag-and-drop each element to anywhere on the grid. Use this to group elements.
  • Click on each element to toggle the foreground and background colours, making it easier to see information you are interested in.

Configuration

  • Enable Bluetooth on your computer.
  • On your PM: More Options -> Turn Wireless On.
  • Click the Start pm5-detail! link above.
  • Click Connect and accept pairing to the PM.
  • Start rowing!

Application Notes

  • This application is for a single erg only.
  • This application has been tested successfully using the %chrome-web-browser% on Mac OS X.
  • Disable any power save or screen saver settings on your computer.

Configuration

This application works on the configurations listed below. Click on the link that matchines your configuration for a quick setup video.

  • Mac OS X over Bluetooth
  • Mac OS X over USB (requires %c2proxy%)
  • Windows over Bluetooth
  • Windows over USB (requires %c2proxy%)
  • Android over Bluetooth

Application Notes

  • This application is for a single erg only.
  • This application supports multiple ergs.
  • Windows over Bluetooth setups requires Chrome version 70 or greater; make sure to update your %chrome-web-browser%.
  • For multi-erg setups, such as studios or gyms, connect all ergs to a powered multi-port USB hub, and the hub to your computer.
  • Disable any power save or screen saver settings on your computer.
  • Ensure your %chrome-web-browser% is up-to-date.

About

The %ergarcade% is available for personal or commercial use at no cost for use in any setting without restriction. Use this software to help your training, improve your coaching or even setup your own ergometer studio.

Want a custom application, tool or visualisation? Have an idea for an app? Please %get-in-touch%!

%support%

Developers

The %ergarcade% is %open-source-software%. Source code for all applications is available on %github-ergarcade%. Patches and pull requests are welcome!

Author

Dean Fogarty is a software engineer based in Brisbane, Australia. He came up with the %ergarcade% as a way of making coaching and training on the ergometer more engaging, entertaining and fun.

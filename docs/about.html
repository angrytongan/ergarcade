About

The Erg Arcade is available for personal or commercial use at no cost for use in any setting without restriction. Use this software to help your training, improve your coaching or even setup your own ergometer studio.

Want a custom application, tool or visualisation? Have an idea for an app? Please get in touch!

Support The Erg Arcade!

While the Erg Arcade is free to use, it has the same running costs as any other online application. By becoming a patron, your small monthly contribution goes directly to covering running costs, bug fixes, updates and future development.

Developers

The Erg Arcade is Open Source software. Source code for all applications is available on GitHub. Patches and pull requests are welcome!

Author

Dean Fogarty is a software engineer based in Brisbane, Australia. He came up with the Erg Arcade as a way of making coaching and training on the ergometer more engaging, entertaining and fun.
